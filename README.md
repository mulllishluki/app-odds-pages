# app-odds-pages

本仓库用于归档和发布多个独立 HTML 页面。这些页面以静态文件形式存放，不依赖后端服务，可直接通过浏览器访问。

## 项目简介

`app-odds-pages` 是一个静态页面集合仓库，主要用途为：

- 归档各类独立功能或信息展示的 HTML 页面
- 提供可通过 GitHub Pages 或其他静态托管服务直接访问的页面文件
- 保持页面内容与仓库分离，便于版本管理及历史追溯

## 目录结构

```
app-odds-pages/
├── README.md           # 本文件
├── index.html          # 仓库首页 / 导航页（可选）
├── pages/              # 存放所有独立 HTML 页面
│   ├── example-1.html
│   ├── example-2.html
│   └── ...
└── assets/             # 公共资源（CSS、JS、图片等）
    ├── style.css
    └── script.js
```

说明：
- `pages/` 目录为页面主体，每个 HTML 文件为一个独立页面。
- `assets/` 存放页面共享的样式、脚本及图片资源，便于统一维护。
- 根目录下的 `index.html` 可作为页面索引或导航入口（可选）。

## 页面归档说明

- 所有页面均为纯静态 HTML，不包含动态后端逻辑。
- 页面之间相互独立，无数据依赖或跨页通信。
- 页面内容以归档和展示为目的，不涉及实时数据或用户交互。
- 如需新增页面，请将 HTML 文件放入 `pages/` 目录，并确保引用资源路径正确。

## 使用方式

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/your-username/app-odds-pages.git
   ```
2. 直接在浏览器中打开 `pages/` 下的任意 HTML 文件即可查看。
3. 若启用 GitHub Pages，可将仓库发布为静态站点，通过对应域名访问。
